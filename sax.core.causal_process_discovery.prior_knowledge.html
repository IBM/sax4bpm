Prior Knowledge

This module contains implementation of a sax.core.causal_process_discovery.prior_knowledge.PriorKnowledge class. This class should be instantiated with the event log data passed to causal discovery algorithm in case we want this algorithm to take into account specific business-process constraints (such as known time precedence between activities in the process)

class sax.core.causal_process_discovery.prior_knowledge.PriorKnowledge(data: DataFrame)

Bases: object

Prior knowledge representation for a particular event log. This object should be created and passed to the causal dependency discovery algorithm in case the user intends to run the algorithm with prior knowledge. The rules for creation of prior knowledge matrix are described here https://lingam.readthedocs.io/en/stable/tutorial/pk_direct.html
